# Mabinogi Tools

A collection of browser-based utilities for Mabinogi.

## Tools

- **[AH Coupon Calculator](ah-coupon-calculator/)** — Calculate the optimal sales tax reduction coupon for Auction House listings, with VIP support and party split calculations.
- **[Barter Checklist](barter-checklist/)** — Track barter trades across NPCs.

## Usage

Each tool is a standalone HTML file. Open directly in a browser or visit the GitHub Pages site.
